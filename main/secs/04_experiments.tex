\subsection{Ablation Studies}

Use ablations to isolate the contribution of each design choice: remove or
swap one component at a time and report the resulting change relative to the
full method. Keep the ablation table in the same style as the main results
table so readers can compare rows directly. State the compute budget used for
each ablation run so that differences in the numbers are not confounded with
differences in training time.

\subsection{Qualitative Analysis}

Complement the quantitative tables with a short discussion of representative
successes and failure cases. Point to specific examples rather than making
general claims, and note any pattern shared across the failure cases, since
that pattern is often the most useful signal for future work.
