\section{Introduction}

Research projects often need a paper template that looks polished without
mixing project-specific commands into the class file. This repository separates
the reusable layout from the example content so that authors can start from a
small, readable source tree. The first visual module is shown in
\figref{fig:first_visual_module}.

\parahead{Main Features}
The class provides a title panel with authors, affiliations, contributions,
abstract, keywords, links, and contact information. It also configures page
geometry, links, section headings, captions, lists, tables, and bibliography
support.

The template is designed for standard \texttt{pdflatex} and \texttt{latexmk}
workflows. It avoids external font files, which makes the repository easier to
share and submit to arXiv.
